# Hotel Booking Cancellation Prediction

A machine learning web application that predicts whether a hotel booking is likely to be canceled. Built with FastAPI, TensorFlow, and deployed on Render.

## Project Overview

This project uses a neural network model to predict hotel booking cancellations based on various booking and customer features. The model helps hotels identify high-risk cancellations and implement proactive retention strategies.

## Live Demo

The application is deployed on Render 

## Tech Stack

- **Backend**: FastAPI, Python
- **Machine Learning**: TensorFlow, Keras, Scikit-learn
- **Frontend**: HTML, CSS (Jinja2 templates)
- **Deployment**: Docker, Render
- **Data Processing**: Pandas, NumPy
